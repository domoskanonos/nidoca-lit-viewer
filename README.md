# nidoca-lit-viewer

nidoca is a lightweight ui-framework based on
[webcomponents](https://www.webcomponents.org/),
[typescript](https://www.typescriptlang.org/)
and [lit](https://lit.dev/).

visit showcase page: [nidoca.de](http://nidoca.de)

### usage
#### npm
    npm i @domoskanonos/nidoca-webcomponents


#### html
```html
  <script src="https://domoskanonos.github.io/nidoca-lit-viewer/nidoca-lit-viewer.js"></script>
  <style>@import url('https://domoskanonos.github.io/nidoca-webcomponents/webcomponents/nidoca.css');</style>

  <webcomponent-viewer-playground></webcomponent-viewer-playground>
```
